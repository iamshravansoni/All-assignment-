About most common formatting tag:-

  1. Em tag ( <em> ):
    • The <em> tag is used to italic a text
    • The <em> The content inside is typically displayed in italic.
    • There is also a tag used to italic a word : <b>
  2. Stong tag ( <strong> ):
    • This <strong> tag is used to bold a text
    • The <strong> HTML element indicates that its contents have strong importance, seriousness, or urgency.
    • There is also a tag used to bold a word : <i>
  3. Strike tag ( <strike> ):
    • The <strike> HTML element renders text with a strikethrough, or a line through it.
    • Use the element to represent things that are no longer relevant or no longer accurate.
    • There is also a different tag used to strike a word : <del> and <ins>
  4. Mark tag ( <mark> ):
    • This <mark> tag is used to highlight a word.
    • We can use the mark element with the blockquote element to highlight the quoted text.
